## Vision

### 10x Check
The right words for how you feel right now, surfaced automatically — no tabs, no choice, no friction. The system reads your prior session tone, the time, and your history, and surfaces the word set most likely to fit. The tab row becomes a fallback ("not quite — try these") rather than the primary UX. This is Night Mode generalized to all emotional states.

### Scope Decisions

## Key Architecture Decisions
- `resolveTextureSetId(value: unknown): TextureSetId` — single exported function for all fallback logic
- `CLEAR_TEXTURES` dispatcher fires synchronously before animation, before Zustand write
- `setTextureSetId(id)` fires in `handleSetChange` (not `onTransitionEnd`) — persistence is immediate
- Night Mode: conditional render `{!isNight && <TextureSetTabs />}` — no animation, state preserved
- `key={resolvedSetId}` on TagGroup — forces clean remount, no Activity needed
- `Activity` from React: explicitly not used — we WANT state to clear on set switch
